# Overview

This repository is a comprehensive starter kit designed to kickstart your full-stack web applications using the latest technologies. It combines **Next.js 16** for a robust React framework, **Better Auth** for seamless and secure authentication, and **Drizzle ORM** for efficient database management with TypeScript support. Perfect for developers looking to build modern, scalable apps without reinventing the wheel.

## Key Features

- **Next.js 16 Integration**: Leverages the newest features of Next.js, including improved performance, server components, and app router for a cutting-edge development experience.
- **Better Auth Setup**: Pre-configured authentication system with Better Auth, supporting various providers (e.g., OAuth, email/password) and secure session management out of the box.
- **Drizzle ORM**: Type-safe database interactions with Drizzle, making it easy to define schemas, run migrations, and query data efficiently. Supports popular databases like PostgreSQL, MySQL, and SQLite.
- **TypeScript Ready**: Fully typed codebase to catch errors early and improve developer productivity.
- **Modular Structure**: Organized folder structure with examples for pages, API routes, components, and utilities, making it easy to extend and customize.
- **Development Tools**: Includes ESLint, Prettier, and other dev tools for code quality. Ready for deployment on platforms like Vercel.

## Getting Started

1. Clone the repository: `git clone https://github.com/Abdullah963039/next16-betterauth-drizzle-starter-kit.git`
2. Install dependencies: `npm install` (or use yarn/pnpm/bun)
3. Set up your environment variables (e.g., database URL, auth secrets) in a `.env` file.
4. Run the development server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000) to see your app in action.
